# CodeMirror

Does 3 things:

1. Adds a codemirror to the textarea code boxes in ACF
2. Gives shadows to table rows
3. Puts MaterialIcons CSS in the head. (https://cdn.jsdelivr.net/npm/@mdi/font@5.8.55/css/materialdesignicons.min.css)
